Topic 1: Introduction to C# and .NET 8
1. Overview of the .NET Ecosystem and Versions
What is .NET?

Imagine you want to build a house (your program). You need tools, materials, and a set of rules to follow. That’s what .NET provides: a framework with tools and libraries to build applications.

It’s created by Microsoft and supports multiple platforms (Windows, Linux, macOS, Android, iOS, Web).

Evolution of .NET:

.NET Framework (2002–2019): Windows-only, older technology.

.NET Core (2016–2020): Cross-platform, faster, modern.

.NET 5, 6, 7, 8 (2020–present): Unified platform — one framework for everything (desktop, web, mobile, cloud, AI).

Why is .NET important?

It saves you time by providing ready-made libraries (like Lego blocks).

It’s secure, fast, and widely used in enterprise applications.

💡 Analogy:  
Think of .NET as the "operating system for developers." Just like Windows helps you run applications, .NET helps you create applications.

2. Benefits of C# 12 Features in .NET 8
C# is the programming language you use with .NET. Each new version adds features that make coding easier, safer, and more powerful.

Key Features in C# 12:
Primary Constructors

Lets you define constructor parameters directly in the class declaration.

csharp
public class Student(string name, int age)
{
    public string Name { get; } = name;
    public int Age { get; } = age;
}

var student = new Student("Alice", 20);
Console.WriteLine($"{student.Name}, {student.Age}");
👉 Cleaner code, fewer lines.

Default Lambda Parameters

Lambdas are small functions. Now you can give them default values.

csharp
Func<int, int, int> add = (x, y = 10) => x + y;
Console.WriteLine(add(5)); // Output: 15
👉 Useful when you don’t always want to pass all arguments.

Collection Expressions

Easier way to create lists/arrays.

csharp
int[] numbers = [1, 2, 3, 4, 5];
👉 Shorter syntax, more readable.

Required Members

Forces you to initialize certain properties when creating an object.

csharp
public class Employee
{
    public required string Name { get; init; }
    public int Age { get; init; }
}

var emp = new Employee { Name = "John", Age = 30 }; // ✅ Works
// var emp = new Employee { Age = 30 }; ❌ Error: Name is required
👉 Prevents mistakes by ensuring important data is always set.

3. Setting up .NET SDK and Visual Studio Code/Visual Studio
Before you can write C# programs, you need the right tools.

Step 1: Install .NET SDK
Go to dotnet.microsoft.com.

Download the latest .NET 8 SDK.

This gives you the compiler and runtime to build/run apps.

Step 2: Choose Your Editor
Visual Studio Code (VS Code):

Lightweight, cross-platform.

Needs extensions (C# extension).

Visual Studio (Windows only):

Full IDE with debugging, GUI design, database tools.

Best for enterprise-level projects.

👉 Beginners often start with VS Code because it’s simple.

Step 3: Create Your First Program
Open terminal/command prompt.

Run:

bash
dotnet new console -n HelloWorldApp
cd HelloWorldApp
dotnet run
Output:

Code
Hello, World!
💡 Explanation:

dotnet new console → Creates a new console project.

dotnet run → Compiles and runs the program.

✅ Summary of Topic 1
.NET is Microsoft’s framework for building apps across platforms.

C# 12 + .NET 8 introduces powerful new features (primary constructors, required members, etc.).

To start coding, install the .NET SDK and use VS Code/Visual Studio.

Your first program is just a few commands away: dotnet new console → dotnet run.
